# 自动公式识别的 macos 小工具

## 目前已经实现：

### 功能一：Latex 公式识别

使用快捷键：command + shift + A 调出截图工具
流程：
1. 快捷键：command + shift + A 截图
2. 调用大模型自动识别公式
3. 将识别结果复制到剪切板，并跳动Dock图标通知用户已复制

#### 设置功能

### 功能二：LaTeX 公式转 MathML (word) 公式

使用: 点击软件界面的 “复制MathML(word)” 按钮即可复制到剪贴板

### 功能三：校园网自动登录

使用快捷键: command + shift + 1
